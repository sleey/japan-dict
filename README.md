# japan-dict

Prepared Japanese dictionary release assets for Wakaru.

These assets are consumed by Wakaru for optional offline dictionary lookup.

Latest assets:

- `latest.json`
- `wakaru-dictionary.zip`

Dictionary data is derived from JMdict and KANJIDIC2 by the Electronic Dictionary Research and Development Group. See [ATTRIBUTION.md](ATTRIBUTION.md).
